String Methods

Métodos slice(start, end), substring(start, end) y substr(start, length).

El método "slice" extrae un parte de un string, se le pasan dos parámetros desde donde queremos empezar y hasta donde queremos extrar la porción de string. El método "substring" tiene la misma la función solo que este no acepta núimeros negativos como parámetros. Y el método "substr" en los parámetros le indicamos desde donde queremos empezar y el segundo parámetro la longitud de string que queremos quitar.
